# AGR-PHP-001 — Pure PHP SaaS Stack

When building a SaaS application without Laravel, use the following stack and architecture:

* PHP 8.3+
* Composer + PSR-4 Autoloading
* MVC Architecture
* Service Layer
* Repository Pattern
* API-first Design

Frontend:

* Tailwind CSS
* Alpine.js
* Fetch API

UI & Components:

* Tabler Dashboard
* Grid.js
* ApexCharts
* Lucide Icons
* SweetAlert2 or Notyf

Database & Cache:

* MySQL 8+
* Redis

Project Structure:

* Controllers
* Models
* Services
* Repositories
* Middleware
* Helpers

Requirements:

* Modular Architecture
* Scalable Design
* Low Resource Usage
* Shared Hosting to VPS Compatibility

Avoid writing framework-like features repeatedly. Keep the codebase clean, modular, and maintainable.
